# SportsLogicAI

A lightweight NFL + NBA betting logic app designed to run on Android as an APK.

### Tech Stack
- HTML
- CSS
- JavaScript

### How to Use
1. Open `index.html` to run the web app.
2. Tap "Get Prediction" to generate a sports pick.

### APK Build Support
This repo is designed to be wrapped using Apache Cordova to generate an Android APK.
